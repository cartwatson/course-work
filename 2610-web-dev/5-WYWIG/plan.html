Requirements Specification

The program will provide the user with a UI that allows them to input their weight (as a number) and the unit of their weight, press a convert button, and see how much their weight is worth in the current price of gold

The program will provide this UI through a single page application

The unitconversion will take place in the django app, all hardcoded values are in a migration

The price of gold comes from querying the NASDAQ through their API

Challenges will include; breaking down the HTTP response for unit conv, and querying NASDAQ. The rest has already been implemented in other projects and should be familiar
