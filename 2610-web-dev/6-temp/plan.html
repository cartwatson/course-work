Requirements Specification

Program must make a call to the IP API to find the user's latitude and longitude

Program must make a call to the weather API, using the user's latitude and longitude

Call should return current conditions

Results should come back in imperial units

While results are pending, loading indictor is displayed, removed when results are returned

Make a call to the weather API for the 5-day 3-hour forecast

All results displayed in their own divs

Divs are color coded to "defualt", "likely", and "unlikely"

Display a count for the number of each of the likelyhoods

System Analysis

Vue needs to get the user's IP address

The IP address is used to get the users latitude and longitude from FreeGeoIP

The latitude and longitude are provided to OpenWeatherApp to get current weather conditions

The latitude and longitude are provided to OpenWeatherApp to get 5-day 3-hour forecast

VueJS handles all of that information and displays it to the user through manipulating the DOM
